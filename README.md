# Sistema Automatizado de Trade (TradingView)

> Um projeto pessoal desenvolvido em **PineScript**, com o objetivo de automatizar estratégias de trade e gerar ordens de compra/venda com base em indicadores técnicos.

---

## Funcionalidades
- Análise de indicadores técnicos (Médias Móveis, Keltner Channel)
- Geração de sinais (ordens) automáticos de compra e venda
- Testes de backtesting no TradingView
- Configuração de ordens automáticas
- Foco em lógica, desempenho e aprendizado

---

## Aprendizados
Durante o desenvolvimento, aprofundei meu entendimento sobre:
- **Lógica de programação e estrutura condicional**
- **Tomada de decisão automatizada**
- **Análise de dados financeiros**
- **Boas práticas de otimização de código**
- **Comunicação com webhooks das corretoras**
